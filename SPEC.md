## 0. 這份原型只回答一個問題

> 「困敵 → 熟成 → 引爆」這場即時賭注，在**走位動作**裡到底爽不爽？
> 而「掙脫頻率」這顆主旋鈕，該落在「賭場」和「格鬥場」之間的哪個位置？

其他一切（關卡、成長、社交）**都不做**。這是一次手感驗證，不是一款遊戲。判準只有一個：滑動掙脫頻率時，你能不能親手摸到那個「同時有喘息、又有張力」的甜蜜點。

## 1. 核心迴圈

熱拍（反射）→ 冷拍（下注）→ 引爆（爽點）→ 可能崩盤（切回熱拍）：

1. 敵人生成、遊走 → 玩家**吐泡泡困住**牠們（動作壓力：熱拍）。
2. 困住越多，畫面越安靜 → 進入**冷拍**：被困泡泡開始熟成，價值↑、掙脫機率↑、**獎勵變好**。
3. 玩家判斷時機，**身體碰上泡泡引爆** → 觸發鄰近連鎖結算（爽點）＋依 Tier 領取獎勵（§5.5）。
4. 若有敵人**掙脫**：暴走 + 震鬆鄰居 → 可能連環崩盤，瞬間切回熱拍。

關鍵設計：**沒有「普通擊殺」。** 殺敵的唯一方式就是「困住再引爆」，所以每次擊殺都是一次下注。v0.2 的 Tier 獎勵讓「等多久」不只賭分數，還賭**你現在最缺的資源**（血？火力？保命？）。

## 2. 實體與狀態機

### 2.1 玩家 Player（v0.2 改版）
- 可左右移動、跳躍。
- 面向方向吐泡泡（發射體）；持有**雙泡能力**時一次吐兩顆（§5.5）。
- **集氣（v0.3）**：按住吐泡鍵開始集氣，約 0.8s 集滿；放開時若已集滿，射出**強力泡**（較大、較快）。強力泡困住的敵人進入「掙脫鎖定期」（§2.3）。未集滿放開＝普通泡泡。集氣中可正常移動跳躍，頭上顯示集氣條。
- 身體碰到「已困泡泡」即戳破 → 引爆。
- **血量制**：`hp`，滿血 = `maxHp`（預設 5，可調 1–10）。碰到自由 / 暴走敵人扣 1 血，並獲得 2 秒無敵（閃爍，同重生保護）。`hp = 0` → 死亡，本局結束。
- **無敵星狀態**（§5.5 紅泡獎勵）：期間不會受傷；身體碰到自由 / 暴走敵人時**自動把牠變成泡泡**（Tier 0，不是消滅）。彩虹閃爍提示，倒數結束前 2 秒加速閃爍。

### 2.2 敵人 Enemy（狀態機，同 v0.1）
```
FREE（自由遊走）
  │  被玩家泡泡命中 / 被無敵星玩家碰到
  ▼
TRAPPED（被困，開始熟成 Tier 0→3）
  │                          │
  │ 掙脫擲骰成功              │ 玩家碰觸 → 引爆
  ▼                          ▼
ENRAGED（暴走：加速、短暫無敵）   DEAD（計分 + Tier 獎勵）
  │  被玩家泡泡再次命中 / 被無敵星玩家碰到
  ▼
TRAPPED（重置回 Tier 0）
```

### 2.3 泡泡 Bubble（同 v0.1）
- **敵泡泡**：困住敵人的泡泡，緩慢上浮 + 隨機晃動，帶熟成 Tier 顏色。無敵星自動困敵產生的泡泡有 0.6s 引爆保護（避免碰觸瞬間就爆）。
- **發射泡泡**：玩家吐出的空泡泡，飛行中，命中第一個 FREE / ENRAGED 敵人即轉為敵泡泡。
- **強力泡（v0.3）**：集氣射出的發射泡泡。困住敵人時，該敵泡泡獲得 `powerLock` 秒鎖定：**期間不擲掙脫骰（含震鄰的即時擲骰），熟成照常推進**——等於一段無風險熟成窗，這就是集氣付出的時間換來的報酬。`powerLock` 拉到滑桿最大值＝永不掙脫（簡單難度預設）；最難難度只鎖 3 秒。鎖定中的泡泡有白色外環標記，解鎖瞬間閃一下提示。

## 3. 熟成與引爆數值（同 v0.1）

每個敵人 `baseValue = 10`。被困後隨時間推進 Tier：

| Tier | 顏色 | 進入時間 (s) | 價值倍率 | 掙脫機率 (/秒，基準) | 視覺 | 引爆獎勵（v0.2 新增，見 §5.5） |
|------|------|------------|---------|--------------------|------|------|
| 0 醞釀 | 綠 | 0.0 | ×1   | 0%  | 平穩 | 無（純分數） |
| 1 半熟 | 黃 | 1.5 | ×2   | 4%  | 輕微晃動 | 補血 +`healAmount` |
| 2 熟成 | 橙 | 3.0 | ×3.5 | 14% | 明顯抖動 | 雙泡能力 `dualDur` 秒 |
| 3 過熟 | 紅閃 | 4.5 | ×6  | 34% | 快速閃爍 + 抖 | 無敵星 `starDur` 秒 |

- 進入時間除以 `ripenSpeed` 旋鈕（越大越快熟）。
- 掙脫機率乘上 `escapeRate` 旋鈕（**主旋鈕**），每 0.2s 擲一次：
  `p_tick = 1 - (1 - p_sec × escapeRate) ^ 0.2`
- 單顆引爆價值 = `baseValue × tierMult`（再乘連鎖倍率，見 §5）。

> v0.2 之後這張表是**雙重駝峰**：分數期望值的駝峰之外，獎勵本身也在誘惑你多等一個 Tier——黃就收是保守打法，撐到紅是高風險拿保命神器。

## 4. 掙脫與連環崩盤（tail risk，同 v0.1）

- **掙脫**：TRAPPED 敵人每 0.2s 依當前 Tier 擲骰（§3）。成功 → 轉 ENRAGED：移動速度 ×1.6、0.5s 無敵、朝最近玩家衝。
- **震鄰崩盤**：一旦有敵人掙脫，對 `cascadeRadius`（預設 90px）內**所有**敵泡泡：
  - 熟成計時器立即前進 `cascadeStrength × 1.0s`；
  - 追加一次即時掙脫擲骰。
- 效果：**緊密的叢集 = 大連鎖，也 = 大崩盤。** 風險與報酬共用同一份幾何，這是刻意的。

## 5. 連鎖引爆與計分（同 v0.1）

採**鄰近引信模型**：玩家碰觸引爆一顆泡泡 → 衝擊波戳破 `popRadius`（預設 70px）內所有敵泡泡，BFS 擴散。`N` = 連鎖總數。

| 連鎖數 N | 倍率 |
|---------|------|
| 1 | ×1 |
| 2 | ×2 |
| 3 | ×4 |
| 4 | ×7 |
| 5 | ×11 |
| 6+ | ×16 |

**單次連鎖總分** = Σ(每顆 `baseValue × tierMult`) × `chainMult(N)`。`chainStrength` 縮放「額外倍率」（N=1 恆為 ×1，保護休閒下車口）。

### 5.5 Tier 引爆獎勵（v0.2 新增）

引爆結算時，**每一顆**被戳破的泡泡依其當下 Tier 追加獎勵：

| Tier | 獎勵 | 說明 |
|------|------|------|
| 0 綠 | 分數（＋`greenHeal` 補血） | `greenHeal` 預設 0＝純分數；簡單難度設 1，讓「困了就戳」自帶續航 |
| 1 黃 | **補血** | `hp += healAmount`（上限 `maxHp`）。缺血時黃泡就是你的目標 Tier |
| 2 橙 | **雙泡能力** | `dualDur` 秒內一次吐兩顆泡泡（重複取得＝重置倒數，不疊加） |
| 3 紅 | **無敵星** | `starDur` 秒瑪利歐式無敵：不受傷、碰到敵人自動困成泡泡（不是清除）。重複取得＝重置倒數 |

> 設計意圖：獎勵綁 Tier 而不是掉落物，讓「等到幾分熟才收」多一層**資源決策**。紅泡最危險（34%/s 掙脫）但給的是最強的保命+進攻工具，貪婪有了第二種理由。

## 8. 視覺與美術（v0.2：像素素材）

- **風格**：pixel art。`imageSmoothingEnabled = false` + nearest-neighbor 放大，全素材以 base64 內嵌，維持單一 HTML 檔。
- **素材來源（v0.3.1）**：
  - 玩家小龍 = **概念圖 pixel-by-pixel 轉檔**（v0.5：`asset_raw/dragon_ref.png` → `tools/convert_ref.mjs` → 32×32 兩幀。管線：邊界 flood-fill 去背（含全透明背景）、被輪廓包住的透明洞視為被 key 掉的白色補回、每目標格取來源區塊眾數色、全域調色盤貪婪合併）；敵人 = `tools/pixelart.py` 自繪。行走兩幀 8fps；**暴走 = 紅色調色盤置換**，使用者上傳單張素材時才退回程式紅染
  - **Launcher 教學頁**（v0.5）：開場覆蓋層——核心循環、集氣、四色泡泡獎勵表、操作鍵，配「簡單／中等／最難」三鍵開始；`M` 隨時返回
  - 血條 = `heart`、無敵星圖示 = `sun`、泡泡外框 = `particle_circle_outline`（依 Tier 程式著色）、平台 = `grass`、牆與地板 = `brick_wall`，取自 [KAPLAY](https://github.com/kaplayjs/kaplay) crew sprites（MIT License, © KAPLAY Team）
- **素材槽保留**：面板可上傳圖片即時替換玩家 / 敵人 / 暴走敵 / 泡泡；squash & stretch、Tier 著色、閃爍照常套用。正式美術進場只要換圖，不動邏輯。
- **回饋**：大連鎖時畫面震動 + 短暫定格；掙脫瞬間紅色閃光；無敵星彩虹閃爍。這些是「爽」和「痛」的體感來源，比美術重要。

- 配額 = 該關生成的敵人總數；全部困住引爆（唯一擊殺方式）即過關。掙脫不減配額（重困再殺即可），**配額守恆**：已生成 ≡ 已殺 + 場上 + 預告中（§13 實驗逐步驗證）。

- 生怪：spawn 點先閃紅色「!」0.7 秒，落地後 0.6 秒虛體（半透明、雙向不碰撞）——引爆區在上層，不被無預警砸中。

## 10. 驗收條件

v0.1 的五條照舊：

1. **「再等一秒」的誘惑真實存在**：紅閃泡泡讓你猶豫該不該收手。
2. **趕成一叢再引爆的大連鎖，有 jackpot 感**。
3. **崩盤感覺是「活該」不是「隨機」**：抖動與閃爍有把危險預告出來。
4. **主旋鈕掃描測試**：`escapeRate` 從 0.3 拉到 1.8，遊戲能從「像西洋棋」滑到「像手速格鬥」。
5. **休閒下車口成立**：完全不玩下注（困了就馬上戳）也能正常遊玩。

v0.2 追加：

6. **獎勵改變等待決策**：缺血時你會刻意等黃、想清場時會賭紅——Tier 選擇隨局勢改變，而不是永遠等最高分。
7. **無敵星是爽的但不是破圖的**：`starDur` 預設下，星星時間追著敵人跑很爽，但困出來的泡泡叢反而堆高了之後的崩盤風險（風險守恆）。
8. **難易度預設有效**：三顆按鈕按下去，10 秒內就能感覺到「這是另一款遊戲」。

v0.3 追加：

9. **集氣值得**：0.8 秒的集氣曝險（不能吐普通泡）換 `powerLock` 的無風險熟成，在中等難度下「該不該集氣」是一個真實的節奏選擇。
10. **-1 不停生會逼出引爆節奏**：`maxEnemies = -1` 時，不清場就會被淹沒，冷拍被自然壓縮。

## 13. 通關可行性實驗（v0.4）

**方法**：headless Chromium 直驅固定時間步（快轉 ~百倍），反應式 bot 扮演「不會集氣、閃避普通的休閒玩家」（策略：困了快戳、保持距離射擊、貼近就逃、無敵星衝撞）。每步驗證配額守恆不變量。卡關偵測 = 60 秒無擊殺。

**結果**（每難度 2 seeds + 代理實驗）：

| 實驗 | 結果 |
|------|------|
| 簡單 × 休閒 bot | **全 5 關通關**（另一 seed 打到 L5 158/160，血線全程 ≥5）|
| 中等 × 休閒 bot | 死於 L1–L3（bot 閃避差，血線是死因，非配額）|
| 中等 × bot+5 血（閃避力代理） | 達 L3 |
| 最難 × bot+7 血 | 達 L5（第 4 關 92s 過，10 連鎖）|
| 配額守恆不變量 | **全部 run 無破缺**（不存在「殺不完」的結構性死局）|

**結論**：簡單＝休閒保證可通；中等＝需要基本閃避與集氣運用（bot 不會的兩件事）；最難＝給高手，可達但殘酷。實驗過程逼出的修正見 v0.4 變更摘要第 4 點（預告生怪、敵人跳躍、泡泡帶高度——其中泡泡帶高度修掉了真正的死局）。

## 14. 子彈時間 × 連鎖回充飛輪（v0.6，設計案：`docs/bullet_time_dlc.md`）

### 14.1 兩條時鐘（鐵律一的實作核）

BT 期間每個固定步長 `dt` 拆成三份：

| 時鐘 | 倍率 | 管什麼 |
|------|------|--------|
| 世界鐘 | `bulletScale`（0.3） | 敵人移動/AI、發射泡泡、泡泡漂浮晃動、生怪計時、無敵星/雙泡倒數 |
| 玩家鐘 | `btPlayerScale` | 玩家移動、跳、射擊冷卻、集氣（簡單=1.0 主角全速；中等/最難=同世界鐘） |
| 風險鐘 | `riskClock`（1.0，下限 0.5） | **熟成 age、掙脫擲骰節奏、強力泡鎖定倒數** |

湧現效果（刻意保留）：BT 裡泡泡相對你的動作**熟得更快**——子彈時間同時是精算放大鏡與熟成加速器，「該不該再等一秒」的張力被放大而非消除。掙脫骰照真實節奏擲，但掙脫出來的暴走敵移動吃世界鐘（事件照發生、你來得及反應）。

### 14.2 能量經濟（防 easy mode 化）

- 按住 `L` 進入（門檻 0.5s 防抖），**每真實秒消耗 1s**；放開或歸零即退出。
- 上限 `energyMax`（預設 5s）——打得再好也囤不成常駐慢動作。
- 回充只來自連鎖（引爆瞬間）：N=1 **+0**；N=2 +0.5s、3 +1.0s、4 +1.6s、5 +2.3s、6+ +3.0s，×`chainEnergy`。
- 簡單難度：開局送 `btGift`=3s，之後每關開始補到至少 3s（休閒保底）；中等/最難 0 起步，只靠飛輪。
- HUD：左上心心下方的藍色能量條（標 `L`），啟動中發光。頭上維持最多兩條 bar 不變。

### 14.3 BT 內操作

- 重力暫停：`AD` 左右、`WS` 上下緩慢漂移（速度吃玩家鐘），**穿過單向平台**（牆與地板仍實體）；`空白` = 向上衝刺。
- 可以吐泡泡/集氣（速度吃玩家鐘）；**引爆仍須身體碰觸**（鐵律三）。
- 退出瞬間重力恢復。

### 14.4 視覺（鐵律二：全自動）

進入 0.25s 漸進、退出 0.15s 回正：~20° 剪切傾斜＋輕微壓扁＋微放大（讀起來像場景翻成 3D，但操作軸與碰撞完全不變）、平台/地板畫出厚度側邊、冷藍色調、粒子吃世界鐘。玩家不控制鏡頭。

### 14.5 驗收追加

11. **BT 裡下注張力仍在**：慢動作中紅泡照樣讓你猶豫（風險鐘照跑）。若變無腦等紅，鐵律一沒守住。
12. **全速是主體**：正常對局 BT 時間佔比 < 1/3；能量見底被丟回全速混戰的「窒息感」要真實存在。
13. **簡單版單手爽感**：主角全速＋世界慢＋3s 保底，休閒玩家能用 BT 逃命與撿引爆，而不需要理解飛輪。

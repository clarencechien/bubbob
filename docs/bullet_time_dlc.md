# 強化版泡泡龍 — DLC 設計案:子彈時間 × 連鎖回充飛輪

> 本案是主規格 `bubble_bobble_A_prototype_spec.md`(v0.1)的手機導向擴充。
> 它**取代**了 v0.1 的兩處:主旋鈕從 `escapeRate` 換成子彈時間相關參數;原本構想中的「主動傾斜規劃視圖」被子彈時間吸收。

---

## 1. 從哪裡走到這裡(recap)

一路的取捨,依序是:

1. **起點**:想混「泡泡龍 + 魔法氣泡」。先發現兩者方向相反(射擊清空 vs. 堆疊別爆頂),平均混會四不像。
2. **釐清**:「泡泡龍」其實是兩款遊戲——1986 動作平台本傳(Bub/Bob,困敵踩破)vs. 益智泡泡龍(射擊三消)。《砂糖地下迷宮》是本傳的 roguelite 最新作。
3. **定方向**:選**動作本傳 × 魔法氣泡連鎖**——這條沒人做過。困敵 → 泡泡帶色聚叢 → 引爆連鎖。
4. **找脊椎**:泡泡是「多義物件」(武器/平台/物理體/蓄能電池),續作都只放大其一。選 **A 軸:蓄能引爆**——每次擊殺都是一場「停利 vs. 讓利潤奔跑」的即時下注。roguelite 已被砂糖地下迷宮吃掉,不走。
5. **驗證可行性**:動作遊戲能不能邊走位邊下注?能——靠**熱拍(反射困敵)/ 冷拍(安靜下注)** 的呼吸節奏錯開,option 活在「感知」不在「計算」。彈幕射擊的 chain 計分(怒首領蜂)是先例。
6. **寫成 spec v0.1**:玩家、敵人狀態機、熟成 Tier 表、震鄰崩盤、鄰近引信連鎖、五顆調校旋鈕。
7. **美術**:泡泡程式畫(不貼圖)、角色 squash/stretch、敵人壓冷色把暖色階讓給泡泡熟成染色。
8. **手機 / 維度切換**:否掉「重力感應控角色」(漂移毀精準)。維度切換必須改變「你怎麼讀關卡」,不能只是好看。
9. **收斂到本案**:**子彈時間**——按著進入慢動作、場景翻成斜角類 3D、可閃可引爆,能量靠連鎖回充。

## 2. 核心機制:子彈時間 × 連鎖回充飛輪

### 2.1 飛輪迴圈

```
全速混戰(熱拍)
   │  困住一叢、想精算下注
   ▼
按下 → 進入子彈時間(慢動作 + 鏡頭自動微傾成斜角)
   │  在慢動作裡:閃暴走敵、走到叢集、碰觸引爆
   ▼
引爆大連鎖(爽點)
   │  連鎖回充能量條
   ▼
能量足夠 → 可再開一次子彈時間 ……(飛輪)
```

打得好的人,能量靠連鎖源源回充,幾乎能維持在慢動作裡行雲流水;打壞的人能量見底,被丟回全速的混亂自己收拾。**技術與資源綁成同一個飛輪,而且獎勵的正是我們想獎勵的行為(佈局引爆)。**

### 2.2 三條不可違反的設計鐵律

這三條若破,整套機制當場死亡:

**鐵律一:子彈時間買的是「反應」,不是「免死」。**
慢動作期間,畫面時間慢,但熟成與掙脫的**遊戲內時鐘照常跑**。紅泡泡在子彈時間裡照樣會在你眼前掙脫暴走——你只是有機會看清楚、手來得及反應。子彈時間**不凍結風險**。這是與市面上所有子彈時間遊戲的根本差異(見 §3.1)。

**鐵律二:3D 旋轉全自動,玩家不主動轉鏡頭。**
按下子彈時間 → 鏡頭自己緩緩推近 + 傾成斜角(紙片翻 3D 的瞬間),幅度小、鬆手自動回正。給剛好夠的縱深感去讀叢集前後關係,但玩家在慢動作裡只做三件事:**走位、閃、碰泡泡**。手機、滑鼠都不用學怎麼轉。強制轉鏡頭會暈、會過載。

**鐵律三:引爆的「移動成本」不能省。**
就算在手機、就算在慢動作,引爆仍是「身體碰觸泡泡」,不是「點一下那顆就炸」。跑到叢集旁邊那段路,是整套機制的脊椎(注意力稀缺)。省掉它 = 退回無腦點消除。

### 2.3 新的調校旋鈕(取代 v0.1 的 `escapeRate`)

| 旋鈕 | 作用 | 決定什麼 |
|------|------|---------|
| `bulletTimeScale` | 慢動作倍率(如 0.2 = 慢到 1/5) | 反應餘裕多大 |
| `energyMax` | 能量條上限 | 單次子彈時間最長多久 |
| `rechargePerChain` | 每次連鎖回充多少能量(隨 N 遞增) | 飛輪的黏性 |
| `energyDrainRate` | 子彈時間中能量消耗速率 | 稀缺程度 |
| `enrageInBulletTime` | 慢動作中掙脫時鐘的相對速度(**不可設 0**) | 鐵律一的守門旋鈕 |

> `enrageInBulletTime` 是最危險的一顆:設 0 就違反鐵律一,整套設計崩塌。它的合法範圍是「比全速稍慢、但絕不停」。

## 3. 賣點分析

### 3.1 最尖的一刀:一款「慢下來反而不變安全」的子彈時間遊戲

子彈時間本身不新(Max Payne、Superhot 都有)。但**在其他所有子彈時間遊戲裡,放慢時間 = 讓你更安全**。本作明確**反過來**:你放慢的是自己的反應,不是危險本身——那顆正在充電的泡泡,慢動作裡照樣會爆給你看。

這個反轉是真正原創的核心:**子彈時間在這裡不是逃生按鈕,是精算下注的放大鏡**。你買的是「看得更清楚、手跟得上」,代價是危險一秒都沒少。這句話一講,懂遊戲的人就知道這不是又一個慢動作特效。

### 3.2 情緒定位:一款關於「貪婪紀律」的動作遊戲

多數動作遊戲賣的是反射神經或力量幻想。本作賣的是**克制與時機**:知道何時該收手、何時該花掉稀缺的子彈時間、何時該讓連鎖再滾一層。整套情緒是「紀律」而非「爽快輾壓」。這是一個少見、且有辨識度的情緒身分,也剛好對上「間接掌控、延遲回報」的設計取向。

### 3.3 心流被做成了字面的系統

「進入狀態、時間慢下來、一切變清楚」——這種主觀的心流體驗,在本作裡是**字面的獎勵迴圈**:你打得越好,遊戲就真的替你把時間放慢、把畫面看清。精通感不是用分數表示,是被**渲染**出來。主題與機制的婚姻異常緊密,這種「感覺 = 系統」的設計本身就是賣點。

### 3.4 正回饋飛輪 = 高光時刻製造機

連鎖回充能量 → 高手能維持在近乎連續的慢動作裡串接大連鎖 → 畫面是一連串「時間凍住、泡泡連環炸開」的鏡頭。這種**可剪輯、可炫技**的高光時刻,是社群傳播的燃料。同時它**自我平衡難度**:強者滾雪球進心流、弱者能量見底被拉回基礎混戰,不需要傳統難度選單。

### 3.5 真正能單手玩的動作手遊(市場楔子)

側捲軸動作平台在觸控上惡名昭彰(要即時又要精準)。子彈時間把最需要精準的那一刻**放慢**,慢動作下用拇指戳、滑、閃,容錯率大幅提高。這讓本作成為少數**真的能直立單手玩、且手感不妥協**的動作遊戲——這是一條實在的市場切入點,不是行銷話術。

### 3.6 美術即機制:紙片角色 + 慢動作斜角

紙片般的平面角色維持高對比、輪廓一眼可讀(適合全速反射);子彈時間的斜角傾斜給出縱深,讓你讀叢集的前後排列(適合下注)。**外觀(傾斜立體透視盒裡的紙片)直接服務玩法(讀叢集深度)**——form follows function,這種一致性讀起來就是「用心設計過」,而非拼裝。

## 4. 誠實的風險(賣點的反面)

- **飛輪失控**:高手若永遠待在慢動作,遊戲變無聊。守門:能量消耗須讓「常駐慢動作」代價夠高,或回充遞減。**全速熱拍在時間佔比上必須是主體,子彈時間是標點不是句子。**
- **慢動作疲勞**:待太久整體節奏變黏。時長與 CD(能量)要調到「珍稀大招」而非「日常模式」。
- **先例洗刷**:Superhot / Max Payne 已建立子彈時間認知。本作的原創性**全押在** §3.1 那個反轉(慢動作不減風險)+ 連鎖回充 + 遊戲內時鐘照跑。行銷上必須把這句差異講清楚,否則會被歸類成「又一個慢動作」。
- **兩套時鐘的實作**:畫面時鐘與遊戲內(熟成/掙脫)時鐘分離,是鐵律一的技術根基,原型就要驗證這件事跑起來直覺不直覺。

---

## 5. 一句話 pitch

> 一款關於**貪婪紀律**的動作遊戲:你能把時間慢下來精算這一注,但慢動作買的是反應、不是免死——那顆泡泡照樣會炸;而你打出的每次連鎖,都在回充讓你再慢一次時間的能量。打得好,你就活在慢動作裡。

---

## 6. 待決 / 下一步

- 能量條制 vs. 固定 CD 制:本案主推**能量條制**(飛輪更有深度),但原型可兩者都試手感。
- 是否進主 spec:確認飛輪好玩後,把 §2 併入 v0.1 成為 §13,並正式退役 `escapeRate`。
- 原型驗收新增一條:**在子彈時間裡,「該不該再等一秒引爆」的張力是否還在?** 若慢動作讓這個決策變得無腦,代表鐵律一沒守住。
